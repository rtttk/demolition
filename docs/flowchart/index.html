服务方操作指南

服务方（拆除团队）在平台上的完整操作流程

前置条件

公司入驻
必须先完成公司资质认证（营业执照等），运营审核通过
团队创建
公司认证通过后，可创建施工团队，设置AB角负责人

服务方完整操作流程

1
浏览接单大厅
在首页查看所有待报价的需求，可按区域、拆除类型筛选
2
查看需求详情
了解拆除类型、面积、地址、预算范围、期望时间等
3
提交报价
填写报价金额、预计工期（天）、施工方案摘要、备注
4
等待报价审核
运营审核报价，通过后需求方可以看到你的报价
5
中标后创建订单
需求方选择你的报价后，你可以创建订单
6
上传合同
订单审核通过后，上传合同文件并填写计划开工日期
7
开始施工
合同审核通过，计划开工日期到达后，订单自动变为"施工中"
8
登记施工日志
每天登记施工进度、工作内容、上传施工照片/视频
9
申请完工
施工完成后，点击"申请完工"，等待需求方验收

运营人员操作指南

运营人员在后台管理系统中的核心审核流程

审核任务总览

公司资质审核
审核服务方的营业执照、资质证书、安全许可证
报价方案审核
审核服务方提交的报价，查看施工方案
订单审核
审核服务方创建的订单，确认项目信息

→
运营审核
→
通过 ✓
或

关键时间节点

订单生命周期中的重要时间点

时间节点 触发条件 自动/手动 后续状态
订单创建 服务方选择报价后创建订单 手动 订单状态: 待审核
订单审核 运营审核订单通过 手动 订单状态: 待签约
合同上传 服务方上传合同文件 手动 合同状态: 待审核
合同审核 运营审核合同通过 手动 订单状态: 待开工
合同状态: 已通过
自动开工 计划开工日期 ≤ 当前日期 自动 订单状态: 施工中
申请完工 服务方点击"申请完工" 手动 订单状态: 待验收
验收确认 需求方点击"验收" 手动 订单状态: 已完成

每日定时任务

⏰ 每日 01:00 自动任务
系统自动检查所有"待开工"状态订单：
如果 计划开工日期 ≤ 今天，自动将订单状态更新为"施工中"
